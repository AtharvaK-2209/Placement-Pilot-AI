┌─────────────────────────────────────────────────────────────────────────────┐
│ 1. LEVELS SYSTEM                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • LevelService (src/services/levelService.ts)
  • 15 configurable thresholds (0 → 43,000 XP)
  • Dynamic level computation from total XP
  • Progress calculation (current XP / next level XP)
  • Level titles: Beginner, Intermediate, Advanced, Expert, Master, Legend
  • Level-up detection

API:
  getLevelState() → { level, currentXP, nextLevelXP, progress, title }
  getLevelForXP(xp) → level number
  checkLevelUp(xpToAdd) → new level or null

Configuration: src/config/gamificationConfig.ts → LEVEL_THRESHOLDS

┌─────────────────────────────────────────────────────────────────────────────┐
│ 2. EXTENDED STREAKS                                                         │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • StreakService (src/services/streakService.ts)
  • ExtendedStreakState type (extends base StreakState)
  • Daily streak (consecutive days)
  • Weekly streak (consecutive weeks with 5+ active days)
  • Monthly streak (consecutive months with 20+ active days)
  • Missed days counter
  • Backward-compatible wrapper methods

API:
  getExtendedStreak() → full extended state
  updateStreakOnDayComplete(date) → updated streak
  getStreakStats() → { current, longest, weekly, monthly, totalActive, missedDays }
  getStreak() → backward-compatible base state
  recordActiveDay() → backward-compatible update
  shouldAwardStreakBonus() → true if streak % 7 === 0

Storage: UserProgress.extendedStreak (optional field)

┌─────────────────────────────────────────────────────────────────────────────┐
│ 3. BADGE SYSTEM                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • BadgeService (src/services/badgeService.ts)
  • 18 badge definitions in 4 categories
  • Lock/unlock state management
  • Automatic condition checking
  • Manual unlock support

Badge Categories:
  [Milestone]    5 badges: First Mission, Roadmap Complete, Future You,
                           Deadline Survivor, Goal Analysis
  [Streak]       4 badges: 7-day, 14-day, 30-day, 100-day warriors
  [Completion]   5 badges: 10, 50, 100, 250, 500 tasks milestones
  [Special]      4 badges: First Week, Level 10, Perfect Week, Execution Intelligence

API:
  getAllBadges() → all badges (locked + unlocked)
  getUnlockedBadges() → unlocked only
  getLockedBadges() → locked only
  unlockBadge(id) → manual unlock
  checkAndUnlockBadges() → auto-check all conditions

Storage: UserProgress.badges (optional array)

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4. XP HISTORY                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • Complete event log in UserProgress.xpLog
  • Immutable append-only structure
  • Date-based filtering via repository methods

Data Structure:
  XPEntry { id, source, amount, earnedAt, description }

API (Repository):
  getXPLog() → all entries
  getXPHistory(startDate, endDate) → filtered entries
  addXP(entry) → append new entry

Storage: UserProgress.xpLog (existing array, unchanged)

┌─────────────────────────────────────────────────────────────────────────────┐
│ 5. WEEKLY GOALS                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • WeeklyGoalService (src/services/weeklyGoalService.ts)
  • Auto-generation every Monday
  • Mission completion tracking (default: 5/week)
  • XP earning tracking (default: 500 XP/week)
  • Progress calculation

Week Bounds: Monday 00:00 → Sunday 23:59

API:
  getCurrentWeekGoal() → current week goal (creates if missing)
  getAllWeeklyGoals() → all historical goals
  updateWeeklyGoalProgress() → recalculate progress
  getCurrentWeekProgress() → { missionProgress, xpProgress, overallProgress }
  getCompletedWeeklyGoalsCount() → total completed weeks

Storage: UserProgress.weeklyGoals (optional array)

┌─────────────────────────────────────────────────────────────────────────────┐
│ 6. MILESTONES                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • MilestoneService (src/services/milestoneService.ts)
  • 9 predefined milestones
  • Unlock tracking with timestamps
  • Badge integration

Milestones:
  • First Login
  • Goal Analysis Complete
  • Roadmap Generated
  • First Mission Complete
  • First Week Complete
  • Future You Generated
  • Deadline Rescue Used
  • Execution Intelligence Used
  • Roadmap Complete

API:
  getAllMilestones() → all milestones
  getUnlockedMilestones() → unlocked only
  unlockMilestone(id) → unlock with timestamp
  isMilestoneUnlocked(id) → check state
  getCompletionPercentage() → % unlocked

Storage: UserProgress.milestones (optional array)
Reward: 100 XP per milestone unlock

┌─────────────────────────────────────────────────────────────────────────────┐
│ 7. GAMIFICATION SERVICE (ORCHESTRATOR)                                     │
└─────────────────────────────────────────────────────────────────────────────┘

Implementation:
  • GamificationService (src/services/gamificationService.ts)
  • Single entry point for all gamification features
  • Coordinates all subsystems
  • Event handlers for task/day/week completion

API:
  awardXP(source, description) → XPAwardResult
  onTaskComplete(title) → XPAwardResult
  onDayComplete(date, title) → { xpResult, streak }
  onWeekComplete(num, title) → XPAwardResult
  
  getLevelState() → LevelState
  getExtendedStreak() → ExtendedStreakState
  
  getAllBadges() → Badge[]
  getUnlockedBadges() → Badge[]
  unlockBadge(id) → Badge | null
  
  getAllMilestones() → Milestone[]
  unlockMilestone(id) → Milestone | null
  
  getCurrentWeeklyGoal() → WeeklyGoal
  getCurrentWeekProgress() → progress percentages
  
  getGamificationState() → complete state snapshot

XPAwardResult:
  { xpAwarded, totalXP, levelUp, newLevel?, previousLevel?, badgesUnlocked[] }

GamificationState:
  { level, totalXP, streak, badges, milestones, weeklyGoal, tasksCompleted }

  import { GamificationService } from '../services';
  const gamification = new GamificationService(progressRepository);

TASK COMPLETION:
  const result = await gamification.onTaskComplete(taskTitle);
  if (result.levelUp) {
    showLevelUpModal(result.newLevel);
  }
  result.badgesUnlocked.forEach(badge => {
    showBadgeToast(badge);
  });

DAY COMPLETION:
  const { xpResult, streak } = await gamification.onDayComplete(date, title);
  showStreakDisplay(streak.currentStreak);

WEEK COMPLETION:
  await gamification.onWeekComplete(weekNumber, weekTitle);

MILESTONE UNLOCK:
  await gamification.unlockMilestone('goal-analysis-complete');
  await gamification.unlockMilestone('roadmap-generated');
  await gamification.unlockMilestone('future-you-generated');
